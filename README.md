# API-Project-Students
This repository includes several student Shark Tank projects developed in Programming for Digital Media. Working in groups, students conducted market research to identify unmet needs in society and brainstormed ideas that could address gaps in the market. They designed methodologies incorporating APIs to improve workflow efficiency and strengthen the implementation of their concepts. From there, they built initial prototypes, gathered user feedback, organized and analyzed that feedback, and revised their code to refine and polish their systems. The projects concluded with engaging final presentations.

These projects included tools such as unit converters, AI-based fashion design recommendation systems, weather-based clothing suggestion apps, and projects that combined multiple AI technologies, such as ChatGPT and Gemini, to generate results.
